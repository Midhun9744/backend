# Human Activity Detection Backend

This project is a backend system for detecting human activities using Wi-Fi signals and deep learning. The system can identify activities like walking, sitting, standing, and falling in real time.

The backend is built using Flask and connects with a Flutter mobile app. It receives Wi-Fi CSI data, processes it, predicts the activity using a trained AI model, and sends the result to the frontend.

---

# Features

- Real-time activity detection
- Fall detection alert system
- Flask API support
- Deep learning model integration
- Frontend communication
